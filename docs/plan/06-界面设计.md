# 06 - 界面设计

## 6.1 设计规范 (Apple风格)

### 配色方案

- **主色调**: `#0ea5e9` (天蓝色)
- **背景色**: `#ffffff` (白色) / `#1f2937` (深色模式)
- **文字色**: `#1f2937` (深灰) / `#f9fafb` (深色模式)
- **边框色**: `#e5e7eb` (浅灰)
- **成功色**: `#10b981` (绿色)
- **警告色**: `#f59e0b` (橙色)
- **错误色**: `#ef4444` (红色)

### 圆角与阴影

- **卡片圆角**: `12px`
- **按钮圆角**: `8px`
- **输入框圆角**: `8px`
- **卡片阴影**: `0 2px 8px rgba(0,0,0,0.1)`
- **悬浮阴影**: `0 4px 16px rgba(0,0,0,0.12)`

### 字体

- **标题**: `-apple-system, BlinkMacSystemFont, 'Segoe UI'`
- **正文**: 14px / 16px
- **小字**: 12px

### 间距

- **内边距**: 12px / 16px / 24px
- **外边距**: 8px / 16px / 24px / 32px
- **组件间距**: 16px

### 动画

- **过渡时长**: 200ms - 300ms
- **缓动函数**: `ease-in-out`
