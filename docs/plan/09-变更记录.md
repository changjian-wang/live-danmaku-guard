## 技术决策记录

### 为什么选择Vue 3而不是React?

**决策日期**: 2026-01-04

**原因**:
1. Vue 3的Composition API与.NET的风格更接近
2. TypeScript支持优秀
3. Naive UI提供了优秀的组件库
4. 学习曲线相对平缓
5. 国内社区活跃，中文文档完善

### 为什么选择SQLite而不是PostgreSQL/MySQL?

**决策日期**: 2026-01-04

**原因**:
1. 轻量级，无需单独安装数据库服务
2. 部署简单，适合桌面应用
3. 性能对于预期数据量足够
4. 支持迁移到其他数据库（EF Core抽象）
5. 适合个人和小团队使用

### 为什么选择Electron?

**决策日期**: 2026-01-04

**原因**:
1. 代码复用 - Web和桌面共享同一套代码
2. 跨平台 - Windows、macOS、Linux
3. Web技术栈 - 前端开发者友好
4. 成熟生态 - VSCode、Slack等都在使用
5. 开发效率高
